How to solve the different tasks

Tasks which can't be solved using the dashboard and the camera, but need separate systems which couldn't be implemented in the dashboard.

Get videostream

  • Download GStreamer on Operator Station (use right bit version and not developer)
  • Get IP address of operator station using ipconfig
  • Change IP in the sh file on pi
  • Execute on PI: sudo service motion stop
  • Followed by: ./startVideo.sh
  • Go on operator station to folder D:\gstreamer\1.0\x86_64\bin (D: then folder)
  • Execute /gst-launch-1.0.exe -e -v udpsrc port=5001 ! "application/x-rtp, encoding-name=JPEG, payload=26" ! rtpjpegdepay ! jpegdec ! autovideosink

Analyse Audio

  • Record Audio using Dashboard
  • cd Desktop/Work/RMRC/dejavu-master/
